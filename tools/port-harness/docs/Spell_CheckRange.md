# Spell::CheckRange

**File:** src/game/Server/Packets/Spell.cpp

## Summary
Spell::CheckRange validates whether the spell's unit target, game-object target, or destination location is within the spell's configured min/max range (with strict vs non-strict tolerance, leeway, spell range mods, and special cases for self-only and combat-range spells), returning a SpellCastResult failure code or SPELL_CAST_OK.
